# 关于考试

**选课建议**

我选的林志杰老师的课，还不错。

**考试形式**(以2019-2020期末考试为例)

这门课是专业选修课，3学分。推荐选。

| 成绩分项 | 成绩分项比例 |
| -------- | ------------ |
| 平时     | 40%          |
| 期末     | 60%          |

大型数据库前面都是比较基础的内容（如果你前面的Mysql学得还不错的话），后面涉及到存储过程和触发器的时候，可能会遇到一些困难。老师人挺好的，书看不懂的话可以去找老师多问问。

倒数一二节课老师会安排一次习题课，来模拟考试，来决定出题的难度。
